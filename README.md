## Visão Geral
 
pInk - converter é um aplicativo que converte arquivos `.cbr` para `.cbz`.
 
## Funcionalidades

- Interface gráfica para desktop
- Seletor de pasta nativo
- Prévia: Antes de iniciar, o app escaneia a pasta selecionada e exibe quantos arquivos .cbr serão convertidos e quantos .cbz serão mantidos.
- Opções de conversão:
    - Remover CBRs após conversão
- Log em tempo real.
- Barra de progresso
- Processamento paralelo multi-thread

## Requisitos

Para extrair os arquivos RAR, o sistema precisa ter **unrar** ou **7z** instalado e disponível no PATH.

## Download
 
Acesse a página de [Releases](https://github.com/pinkcomics/converter/releases) e baixe o executável.
