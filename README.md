## Visão Geral

pInk - reforge é um aplicativo desktop que converte arquivos `.cbr` para `.cbz`.

## Funcionalidades

- Interface gráfica para desktop
- Seletor de pasta nativo
- Prévia: Antes de iniciar, o app escaneia a pasta selecionada e exibe quantos arquivos .cbr serão convertidos e quantos .cbz serão mantidos
- Opções de conversão:
  - Remover CBRs após conversão (preferência salva entre execuções)
- Barra de progresso
- Cancelamento de uma conversão em andamento
- Resumo final da execução (total analisado, transformado, mantido, falhas e tempo total)
- Processamento paralelo multi-thread

## Requisitos

Para extrair os arquivos RAR, o sistema precisa ter **unrar** ou **7z** instalado e disponível no PATH.
